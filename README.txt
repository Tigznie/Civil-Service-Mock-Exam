How to use:
1) Extract the ZIP.
2) Open index.html in a browser (Chrome/Edge recommended).
3) Choose exam type and press Start Exam.

Features:
- Professional: 170 items, Subprofessional: 165 items
- Full coverage categories: ENG, VOC, SPL, ANA, RC, MATH, LOG, GEN
- Randomized every attempt
- No-repeat pool per exam type using localStorage (auto-refresh when exhausted)
- Timer (set minutes; 0 disables)
- Scoring + category breakdown + review
